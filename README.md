# 🔐 Auth System MERN

A simple and secure authentication system built using the MERN stack (MongoDB, Express, React, Node.js). This project includes user registration, login, JWT authentication, and protected routes.

---

## 🚀 Features

- User Registration
- User Login
- Password Hashing (bcrypt)
- JWT Authentication
- Protected Routes (middleware)
- Get All Users (without passwords)
- Error Handling

---

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **Authentication:** JWT (jsonwebtoken)
- **Security:** bcryptjs
- **Environment Variables:** dotenv
